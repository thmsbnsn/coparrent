## What This Folder Is

This is the buyer package for **CoParrent**.

The point of this folder is simple: let a serious buyer understand the asset quickly without having to pull the story out of me line by line.

If this package is doing its job, a buyer should be able to answer five questions fast:

- what the product is
- what is already real
- what still needs to be closed
- what they would actually be buying
- how hard this would be to rebuild from scratch

## How To Use It

- Start with the one-pager for outreach.
- Use the FAQ once a buyer starts asking real questions.
- Use the demo script for a live walkthrough or recorded demo.
- Use the target-buyer list and outreach sequence to run the actual process.
- Use the demo-environment plan before recording or giving access to a buyer.
- Keep the data-room checklist internal until diligence is actually happening.
- Keep the deal-structure guide internal. That one is for negotiating, not marketing.
- Use the personalized outreach drafts when you are ready to start contacting the first wave.
- Use the sell-readiness plan to decide what should be finished before the asset is marketed hard.
